1. Maak een nieuwe Laravel installatie

Plaats de "controllers" in de "app/Http/Controllers" map
Plaats de migrations in de "database/migrations" map
Plaats de models in de "app/Models" map
Plaats de seeders in de "database/seeders" map
Plaats de views in de "resources/views" map
Plaats de images map in de "public" map

2. Stel je databank correct in de .env file
3. Voer het commando "php artisan migrate --seed" uit 
   om de databank aan te maken en te voeden met data
